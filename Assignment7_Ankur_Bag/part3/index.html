Basic Class Selectors

1All Selector ("*")

This will select all the elements on the page and toggle them

2Class Selector (.class)

This will select all the elements with same class and change the color of the background color.

3Element Selector ("element")

This will select all the elements with div tag and will give a border.

4Multiple Selector ("selector1, selector2, selectorN")

This will select all the elements with div tag,p tag and will give a border.
